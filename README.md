# 📌 Sistema de Notificações

API REST desenvolvida em Java com Spring Boot para gerenciamento de mensagens/notificações.

---

## 🚀 Tecnologias utilizadas

- Java 21
- Spring Boot
- Spring Data JPA
- SQL Server
- Docker
- Gradle

---

### 1. Clone o repositório

